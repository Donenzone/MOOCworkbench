You have now arrived at the data schema step. In this step, you'll quickly create a data schema for the input data you've used in your experiment

Step 1. Make a schema with the built-in tools

Before you begin, make sure the column names in your data files adhere to the JSON Table Schema standard. Make sure the headers are alphanumeric and lowercase. Then, to get started run make_schema.py like this:

make schema input_filepath=FILEPATH_TO_DATA

Step 2. Commit and push your schema

If everything went right, you now have a file called schema.json in the folder schema (at the root of your project).

Commit and push this file to your repository

Step 3. Check your schema

Once you've pushed your schema, the workbench will pick it up automatically if it is valid.

Go to the Schema tab in your experiment to check if your schema is added already, and make changes if neccessary.
